# Student-Academic-Performance
This project leverages Excel for Data Cleaning and Transformation and Power BI to analyze and visualize student performance data, providing actionable insights into academic outcomes and engagement patterns. The dashboard integrates multiple visuals to highlight key metrics, distributions, and relationships across the data.
